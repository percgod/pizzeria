# pizzeria

1. Формулировка задачи
   1.1 На какую аудиторию рассчитываем?
   Ядро аудитории (80% заказов):
    · Семьи с детьми (25–45 лет): Живут или работают в радиусе 1–2 км от пиццерии. Им важно: ребенок сыт, цена адекватная, можно взять самовывозом или быстро доставить.
    · Офисные сотрудники (25–35 лет): Обычные обеденные перекусы. Платежеспособные, ценят скорость и вкус.
    · Студенты и молодежь (18–24 года): Работают по акциям, любят две по цене одной и необычные вкусы (с ананасом, сырный бортик и т.д.).
