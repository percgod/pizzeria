Task: Разработка сайта для онлайн-пиццерии PizzaTime с конструктором пиццы и системой заказа

Цели бизнеса

· Конверсия трафика в заказ: 4-5%
· Средний чек: 1 300 ₽
· Целевой трафик: 15 000+ уникальных посетителей в месяч (к концу года)

Основные функции сайта

1. Инструмент конструктора пиццы:
   · Выбор основы (традиционное/тонкое тесто).
   · Выбор размера (25см/30см/35см).
   · Выбор ингредиентов (до 8 позиций с динамическим пересчетом цены).
   · Визуализация пиццы в реальном времени.
2. Конфигуратор заказа:
   · Выбор типа борта (стандартный, сырный, колбасковый).
   · Выбор дополнительных соусов (чесночный, сырный, барбекю).
   · Добавление напитков и закусок к заказу.
3. Упрощенная воронка:
   · Каталог -> Конструктор (опционально) -> Корзина -> Оформление (не более 4 шагов).
   · Интеграция с онлайн-оплатой (карты, СБП, Apple Pay/Google Pay).
   · Автоподстановка адреса через DaData/Яндекс.Карты.
4. Система социального доказательства:
   · Галерея фото реальных заказов.
   · Виджет отзывов с Яндекс.Карт/2ГИС.
   · Счетчик "Заказов сегодня" в реальном времени.

Технические требования

· Производительность: Core Web Vitals ≥ 85, FID < 100ms, LCP < 2.5s.
· Адаптивность: Mobile-first, корректная работа на iOS 13+/Android 9+.
· Интеграции: Facebook Pixel, Google Analytics 4, Яндекс.Метрика, CRM (RetailCRM/AmoCRM), платежные системы.
· Безопасность: HTTPS, защита персональных данных, соответствие 152-ФЗ.

Этапы разработки

1. Прототип воронки и UI/UX конструктора пиццы.
2. Интеграция JavaScript-библиотеки для визуализации пиццы.
3. Разработка бэкенда каталога, корзины и админ-панели.
4. Наполнение контентом (40+ позиций), подключение аналитики.
5. Тестирование (юзабилити, нагрузочное, A/B тесты кнопок).
